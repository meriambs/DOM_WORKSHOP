In this example: querySelector('.item') selects the first element with the class "item" (i.e., "Item 1") and changes its text color to blue. querySelectorAll('.item') selects all elements with the class "item" (i.e., all list items) and changes their background color to light gray. When you run this code, you'll see that querySelector() returns only the first matching element, whereas querySelectorAll() returns a NodeList containing all matching elements.
